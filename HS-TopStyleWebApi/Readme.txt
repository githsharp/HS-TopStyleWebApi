#HS-TopStyleWebApi

# Inlämningsuppgift 1: Web API för TopStyle
## Helena Sveding @Jensen Yrkeshögskola
## Moln-databaser

## Beskrivning
- Bygg backend-delen för företaget TopStyle som säljer kläder och skor online och behöver en webbshop.
- Lösningen skall skrivas med ASP.NET Web api och gå mot en SQL databas som sätts upp i Azure.
- Kommunikationen med databasen skall ske med Entity framework och man skall använda Code first.
- Connectionsträngen skall hanteras via en Azure keyvault.
- Dokumentation ska ske med Swagger.
- Github ska användas för att lagra och versionshantera koden.
